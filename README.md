E-Shop - Full Stack E-Commerce Application
-------------------------------------------
A modern, full-stack e-commerce application built with the MERN stack (MongoDB, Express.js, React-style frontend with vanilla JavaScript) featuring user authentication, product browsing, shopping cart functionality, and order management.

Live Demo
------------------------------------------
Frontend: https://chk-ecommerce.netlify.app
-------------------------------------------
Backend API: https://e-commerce-cgg1.onrender.com
----------------------------------------------
Features
--------------------------------------------
1. **User Features**: User Authentication: Secure signup/login with JWT tokens

2. **Product Browsing**: Browse products with filtering and search capabilities

3. **Shopping Cart**: Add/remove items, update quantities

4. **Order Management**: Place orders and view order history

5. **Responsive Design**: Works seamlessly on desktop and mobile devices

Product Features
----------------------------------------------
1. **Advanced Filtering**: Filter by category, price range

2. **Search Functionality**: Search products by name or description

3. **Sorting Options**: Sort by price (low-high, high-low) and newest arrivals

4. **Product Details**: View detailed product information

Technical Features
-----------------------------------------------------
1. **JWT Authentication**: Secure token-based authentication

2. **Password Encryption**: BCrypt password hashing

3. **RESTful API**: Well-structured API endpoints

4. **MongoDB Integration**: Efficient data storage with Mongoose ODM

Technology Stack
---------------------------------------
Backend
- **Node.js** - JavaScript runtime

- **Express.js** - Web application framework

- **MongoDB** - NoSQL database

- **Mongoose** - MongoDB object modeling

- **JWT** - JSON Web Tokens for authentication

- **BCrypt** - Password hashing

- **CORS** - Cross-origin resource sharing

Frontend
---------------------------------
- **HTML5/CSS3** - Structure and styling

- **Vanilla JavaScript** - Core functionality

- **Axios** - HTTP client for API requests

- **CSS Grid/Flexbox** - Responsive layouts
